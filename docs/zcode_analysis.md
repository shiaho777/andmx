# ZCode UI逆向分析报告

## 概览

ZCode是一个专业的AI编程助手，以极简设计和高性能著称。

## 核心设计理念

### 1. 极简布局
- 三栏结构：侧边栏 | 对话区 | 上下文面板
- 深色主题为主
- 扁平化设计，无过度装饰
- 充分利用空间

### 2. Markdown渲染特点

#### 内联元素
- **粗体**：`**text**` → 加粗显示
- *斜体*：`*text*` → 斜体显示
- `代码`：`` `code` `` → 等宽字体 + 浅色背景
- 链接：`[text](url)` → 可点击链接

#### 块元素
- **标题**：`# H1`, `## H2`, `### H3`
  - 层级分明
  - 适当的行间距
  
- **列表**：
  - 无序：`- item`
  - 有序：`1. item`
  - 支持嵌套
  
- **代码块**：
  ```
  ```language
  code here
  ```
  ```
  - 语法高亮
  - 行号显示（可选）
  - 复制按钮
  - 支持滚动
  
- **引用**：`> text`
  - 左侧竖线装饰
  - 浅色文字

### 3. 代码高亮算法

#### 支持语言
- Kotlin, Java, Python, JavaScript, TypeScript
- Go, Rust, C, C++, C#
- HTML, CSS, JSON, YAML, Markdown
- Bash, Shell

#### 高亮规则
1. **关键字**：if/for/while/class/def → 主题色（蓝色/紫色）
2. **字符串**：`"..."` → 绿色
3. **注释**：`//...` `/*...*/` → 灰色
4. **数字**：123, 0xFF → 橙色
5. **函数名**：识别函数调用 → 黄色
6. **运算符**：+/-/*/= → 白色/浅色

#### 性能优化
- 只高亮可见区域
- 使用AnnotatedString而非多个Text
- 缓存解析结果

### 4. 文件图标映射

基于文件扩展名显示对应图标：

#### 代码文件
- `.kt` → Kotlin图标（紫色）
- `.java` → Java图标（红色）
- `.py` → Python图标（蓝色）
- `.js/.ts` → JavaScript/TypeScript图标（黄色）
- `.go` → Go图标（青色）
- `.rs` → Rust图标（橙色）

#### 配置文件
- `.json` → JSON图标
- `.yaml/.yml` → YAML图标
- `.toml` → TOML图标
- `.xml` → XML图标

#### 文档文件
- `.md` → Markdown图标
- `.txt` → 文本图标
- `.pdf` → PDF图标

#### 其他
- 文件夹 → 文件夹图标
- 未知类型 → 通用文件图标

### 5. 交互动画

#### 消息动画
- **进入**：淡入(200ms) + 上滑(10dp)
- **退出**：淡出(150ms)
- **Easing**：CubicBezierEasing(0.16, 1, 0.3, 1)

#### 工具调用
- **展开/折叠**：Spring动画(stiffness=Low)
- **加载指示器**：CircularProgressIndicator(8dp)
- **状态变化**：颜色过渡(300ms)

#### 列表滚动
- **惯性滚动**：系统默认
- **自动滚动**：平滑动画到底部
- **Over-scroll**：弹簧效果

### 6. 性能优化策略

#### 渲染优化
1. LazyColumn虚拟化
2. key参数避免不必要的重组
3. derivedStateOf减少计算

#### 内存优化
1. 图片懒加载
2. 消息分页
3. 及时释放不可见内容

#### 响应优化
1. 流式渲染（不等全部内容）
2. 异步解析
3. 防抖输入

## 实现建议

### Markdown解析器
采用单遍扫描算法：
1. 逐行读取
2. 识别块类型（标题/列表/代码/引用/段落）
3. 解析内联元素
4. 输出结构化数据

### 代码高亮
1. 正则匹配关键模式
2. 构建AnnotatedString
3. 应用颜色SpanStyle
4. 缓存结果

### 文件图标
1. Map<扩展名, Icon>映射表
2. 快速查找O(1)
3. 默认fallback
